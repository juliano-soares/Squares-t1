----- SQUARES -----

Para executar o programa você irá precisar ter instalado o GCC em seu computador,
pois necessitará executar um programa na linguagem C.

Passo a passo para executar:

1 - Extrair o arquico .rar dentro de uma pasta de sua escolha.

- Vai ter os seguintes arquivos dentro desta pasta
  -> main.c
  -> readme.txt

2 - Abra um terminal de controle demtro desta pasta.

3 - Execute os seguintes comandos no terminal.

> gcc main.c -o squares
> ./squares

4 - Pronto o game irá iniciar no terminal.

5 - O game terá uma opção para ler as instruções.

6 - Para sair do game pode escolher a opção 3 no menu inicial e durante o game
pode digitar 0 e irá para o menu.
